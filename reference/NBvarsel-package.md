# NBvarsel: Variable Selection via Cross-Validated Net Benefit

Performs exhaustive or groupwise (backward elimination) variable
selection for binary outcome prediction models using cross-validated Net
Benefit as the optimization criterion. Supports predictor costs,
restricted cubic splines, interaction terms, permutation importance, and
parallel computation. Includes visualizations for model comparison and
variable importance.
